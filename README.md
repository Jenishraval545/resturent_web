# Culinary Cove Restaurant Website

A simple yet elegant restaurant website built using **PHP** and **CSS**.  
This project demonstrates dynamic navigation, reusable components, and modern styling.

---

## 📂 Project Structure

- **after/** → Main PHP pages (menu, ingredients, mission).  
- **inc/** → Reusable components (header, footer).  
- **styles/** → CSS files for base and custom styling.  
- **images/** → All images used in the project.  

---

## ✨ Features

- **Dynamic Navigation with Active State**  
  Each page sets a `$pagekey` variable, and the header applies an `.active` class to highlight the current page.  

- **Reusable Header and Footer**  
  Shared components included across all pages for consistency.  

- **Modern Styling**  
  - `simple.css` → base styles and reset.  
  - `custom.css` → hover effects, active link styling, image zoom, footer animation.  

- **Content Pages**  
  - *Our Mission* → restaurant philosophy and values.  
  - *Ingredients* → fresh, sustainable, locally sourced ingredients.  
  - *Menu* → starters, soups, main courses, desserts.  

---

## 🛠️ Technologies Used

- **PHP** → dynamic rendering, reusable components.  
- **CSS** → styling, animations, responsive design.  
- **HTML5** → semantic structure.  

---

## 🚀 How to Run Locally

1. Install **XAMPP/WAMP/MAMP**.  
2. Place the project folder `resturent_web` inside the server root (`htdocs` or `www`).  
   - Example: `C:\xampp\htdocs\jenish\resturent_web`  
3. Start the Apache server.  
4. Open in browser: http://localhost/jenish/resturent_web/after/menu.php
